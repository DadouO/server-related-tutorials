- [Ansible lint](https://docs.ansible.com/ansible-lint/)

## Ansible lint usage

Install via pip

```bash
> pip install ansible-lint
```

Then use CLI

```bash
> cd FOLDER OR PRE FOLDER
> ansible-lint
```
